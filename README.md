# Dawak | دواك

**Dawak** is a bilingual mobile pharmacy platform for Lebanon. It helps customers search for medicines, find nearby pharmacies, reserve available products, donate unused medicine, and receive updates from pharmacies.

The project was designed and built as a full mobile application with customer, pharmacy, and admin experiences.

> Source code is private and proprietary. This repository is a public showcase only.

## What Dawak Does

Dawak connects customers with local pharmacies through one simple mobile experience.

Customers can:

- Browse medicines and pharmacy profiles
- Search for available products
- Reserve medicine for pickup
- Donate unused medicine
- Earn points and redeem rewards
- Use the app in Arabic or English

Pharmacies can:

- Manage medicine inventory
- Handle customer reservations
- Verify medicine donations
- Create coupons and offers
- Receive operational notifications

Admins can:

- Review pharmacy onboarding
- Moderate platform activity
- Manage platform-wide announcements

---

## Key Features

- Bilingual Arabic and English interface
- Right-to-left Arabic layout support
- Customer, pharmacy, and admin roles
- Medicine catalog and reservation flow
- Donation and loyalty points system
- Pharmacy operations dashboard
- Dark and light theme support
- Designed for real pharmacy workflows in Lebanon
